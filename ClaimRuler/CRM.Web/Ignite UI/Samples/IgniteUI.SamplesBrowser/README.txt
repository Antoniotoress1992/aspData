--------------------------------------------------------------
    System Requirements
--------------------------------------------------------------

1. Microsoft Visual Studio® 2010 or later
2. ASP.NET MVC 4 (http://www.asp.net/downloads)
3. SQL Server® Express or LocalDB (see 'More on SQL Server Connections' section)


--------------------------------------------------------------
    About the Ignite UI Samples Browser
--------------------------------------------------------------
This application is designed to make it easy for you to learn about
Ignite UI controls and features. Here are tips for finding samples in
the solution:

- HTML5-based samples are found in the \HtmlSamples folder
- ASP.NET MVC-based samples have their own code files in 
  the \Controllers and \Views folders and follow the ASP.NET MVC conventions

For help, visit: http://www.infragistics.com/help/


--------------------------------------------------------------
    More on SQL Server Connections
--------------------------------------------------------------
While the majority of the samples contained in the samples browser run
without any specific server-side code, this samples browser uses Microsoft 
SQL Server for data access with ASP.NET MVC-specific features and examples. 
In all cases, these examples can be accomplished using any persistence technology
and use simple object collections to perform server-side binding.

This project includes connection strings for default Visual Studio 2010 
installations with SQL Express (having instance name SQLEXPRESS) and 
default Visual Studio 2012 installations with LocalDB.

Visual Studio 2012 with LocalDB is configured by default and you can 
comment or uncomment the provided connection strings in the web.config
to switch between the default configurations. For non-default 
SQL configurations, customize the connection strings to match your instance
of SQL server.
